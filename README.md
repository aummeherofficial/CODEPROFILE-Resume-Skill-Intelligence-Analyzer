# CODEPROFILE – Resume Skill Intelligence Analyzer

## 🚀 Project Overview
CODEPROFILE is a rule-based Resume Skill Intelligence Analyzer built using Python.  
It detects technical skills from resume text, categorizes them, and generates a profile strength score.

## 🔥 Features
- Skill detection using keyword matching
- Category-based classification
- Resume strength scoring logic
- Profile level classification
- Skill improvement suggestions

## 🛠 Technologies Used
- Python
- Dictionary mapping
- Set operations
- Rule-based scoring

## 🎯 How It Works
1. User pastes resume text
2. System detects skills from predefined database
3. Calculates score using weighted logic
4. Displays profile level and suggestions
